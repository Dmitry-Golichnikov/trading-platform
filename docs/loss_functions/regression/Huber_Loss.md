# Huber Loss (Smooth L1)

## Назначение
Комбинирует преимущества MSE и MAE: квадратичная ошибка для малых отклонений и линейная для больших. Хорошо работает при наличии выбросов.

## Формула
\( L_\delta = \begin{cases}
\frac{1}{2}(y - \hat{y})^2 & |y - \hat{y}| \le \delta \\
\delta |y - \hat{y}| - \frac{1}{2} \delta^2 & |y - \hat{y}| > \delta
\end{cases} \)

## Особенности
- Параметр \(\delta\) управляет границей между квадратичной и линейной частями.
- Сглаженная версия (`SmoothL1Loss`) доступна в PyTorch.
- Обеспечивает устойчивость к шуму без сильной потери чувствительности.

## Рекомендации
- Использовать при прогнозе доходности, где возможны выбросы.
- Настраивать \(\delta\) через валидацию или использовать автоматический вариант (`beta`).
