# Выход по риск-параметрам

## Назначение
Принудительное закрытие позиций при достижении заранее установленных лимитов риска (дневных, портфельных, VAR, drawdown). Защищает капитал от экстремальных потерь.

## Варианты
- **Дневной лимит убытка**: закрыть все позиции, если день закрывается хуже −X%/валют.
- **Портфельный лимит**: суммарный DD по портфелю превышает порог.
- **Exposure limit**: превышение максимальной экспозиции по инструменту/сектору.
- **VAR/Expected Shortfall**: выход при превышении прогнозного риска.

## Параметры
- `risk_metric`: `daily_loss`, `portfolio_drawdown`, `var`, `es`, `exposure`.
- `threshold`: значение порога (проценты, абсолютные величины).
- `window`: период расчёта (день, неделя, rolling N баров).
- `action`: закрыть все позиции, закрыть конкретный тикер, сократить позу.
- `reset`: условия сброса лимита (например, новый торговый день).

## Рекомендации
- Расчёт лимитов выполнять в real-time (streaming) и хранить в логах.
- Синхронизировать с risk-engine брокера/биржи.
- При тестировании учитывать комиссии и проскальзывание при массовых выходах.
